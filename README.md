# Grow My Therapy – Redesign Version

This branch contains the redesigned version of Dr. Maya Reynolds' website.

## What’s Included in This Redesign

- Updated homepage layout
- SEO-optimized copy based on Dr. Maya Reynolds' profile
- New “Our Office” section (custom creative section)
- Improved typography and spacing
- Warm, calming color palette aligned with therapy brand
- Desktop and mobile responsive design
- GSAP animations for enhanced user experience

## Technologies Used

- Next.js
- Tailwind CSS
- GSAP (for animations)
- Vercel (deployment)

## Purpose

This redesign focuses on:
- Clear messaging
- Emotional connection
- Trust-building design
- Improved visual hierarchy
- Better client conversion experience
